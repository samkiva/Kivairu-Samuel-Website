# Data Architecture Roadmap

This directory contains the business entity logic for the HexSentinel Portfolio.

## Current Models
- `projects.ts`
- `skills.ts`
- `services.ts`
- `experience.ts`
- `blog.ts`
- `currentlyBuilding.ts`
- `testimonials.ts`

## Future Models (Planned for Scale)
These models are documented here to establish a consistent pattern for future portfolio expansion. Please define matching strict interfaces in `types/` before implementation.

- `education.ts` (Degrees, institutions, GPA, relevant coursework)
- `certifications.ts` (AWS, GCP, specialized AI certs, issue dates)
- `awards.ts` (Hackathons, academic honors, industry recognition)
- `publications.ts` (Research papers, arXiv links, conference presentations)
- `talks.ts` (Conference talks, meetups, slides/video links)
- `gallery.ts` (Centralized media/image grid metadata for a dedicated /gallery route)
- `faq.ts` (Frequently asked questions about contracting, consulting, or general inquiries)
